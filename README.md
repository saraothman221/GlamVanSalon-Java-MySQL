# Glam Van - Mobile Salon Management System 

**Glam Van** is a comprehensive standalone desktop application designed to digitize and streamline the operations of mobile beauty salons. Developed as part of a **Software Engineering course**, the system transitions traditional manual salon management into a robust, automated digital platform.

## Project Objectives
The primary goal was to solve scheduling conflicts and operational inefficiencies inherent in mobile services through:
* **Centralized Booking:** Providing a user-friendly interface for seamless service scheduling.
* **Operational Automation:** Reducing manual errors by automating appointment tracking and customer data management.
* **Business Insights:** Generating reports on service popularity and booking trends to support data-driven decision-making.

## 🛠️ Technical Stack
* **Language:** Java.
* **IDE:** NetBeans IDE.
* **Database:** MySQL (used for storing user accounts, appointments, payments, and feedback).
* **Architecture:** Standalone Desktop Application.

## Access Control & User Roles
The system implements Role-Based Access Control (RBAC) with three distinct modules:
1. **Manager:** Manages the service catalog, handles staff information, and analyzes customer feedback.
2. **Staff:** Views personalized appointment schedules and manages professional profiles.
3. **Customer:** Allows for account registration, service browsing, secure booking, and payment processing.

##  Core Features
* **Smart Scheduling:** Integrated date and time validation to prevent double-booking.
* **Payment Processing:** Supports both Cash and Card payment methods with automated status updates.
* **CRM & Feedback:** A dedicated system for managing client profiles and collecting service ratings to ensure quality control.
